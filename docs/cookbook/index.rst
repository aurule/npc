NPC Cookbook
=================

NPC can do a lot of handy things. This guide shows how you can use it to speed up common and less common campaign management tasks.

.. note::

    While the goal is to be able to perform every action through both the cli and the gui, that isn't possible yet. As the gui matures, it will be closer and closer to feature parity with the cli. For now, the cli is the more full-featured program for NPC.
